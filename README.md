## Install

Via Composer

``` bash
$ composer require rikirizkins/demo-composer-akademi-batch-2
```

## Usage

```php
use rikirizkins\HaloDunia;

$haloDunia = new HaloDunia();

echo $haloDunia->getMessage();

/*
  Halo dunia dari ICUBE Academiy batch 2, Good luck!
*/
```
